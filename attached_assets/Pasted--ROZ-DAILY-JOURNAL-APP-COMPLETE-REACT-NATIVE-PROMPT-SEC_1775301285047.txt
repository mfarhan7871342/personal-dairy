===== ROZ — DAILY JOURNAL APP: COMPLETE REACT NATIVE PROMPT =====

[SECTION 1 — PROJECT OVERVIEW]
Build a complete React Native mobile app called "Roz — Daily Journal" using Expo (managed workflow) and TypeScript. This is a private, secure, and addictive personal diary app targeting all age groups. The UI should be beautiful, modern, and animated — inspired by premium journaling apps but better.

App color scheme: Deep purple (#6C63FF) as primary, soft lavender (#EDE9FE) as background, white cards, with dark mode support. Use custom fonts (expo-google-fonts — Inter + Playfair Display for headings).

[SECTION 5 — DATA MODELS]

interface DiaryEntry {
  id: string; title: string; body: string; mood: MoodType;
  photos: string[]; voiceUri?: string; voiceDuration?: number;
  createdAt: string; updatedAt: string; themeId: string;
  isFavorite: boolean; tags: string[];
}
type MoodType = 'happy'|'sad'|'anxious'|'excited'|'calm'|'angry'|'grateful'|'neutral';
interface UserSettings {
  lockType: 'none'|'pin'|'biometrics'|'pattern'; pin?: string;
  reminderEnabled: boolean; reminderTime: string; themeId: string;
  fontSize: 'small'|'medium'|'large'; userName: string; language: 'en'|'ur';
}
interface StreakData {
  currentStreak: number; longestStreak: number;
  lastEntryDate: string; totalEntries: number; unlockedBadges: string[];
}

Zustand stores: useEntryStore (CRUD + queries), useSettingsStore, useStreakStore (auto badge unlock), useMoodStore (stats).

[SECTION 6 — ANIMATIONS + UX]
- Reanimated v3 throughout: FadeInDown stagger on lists, spring scale on mood picker, FAB rotate+scale, badge Lottie confetti, PIN shake on wrong entry, voice waveform sine animation, AI typing 3-dot pulse.
- expo-haptics on all taps and badge unlocks.
- Skeleton loaders on all async operations.
- Swipe-to-delete with 3s undo snackbar.
- Full dark mode via theme/colors.ts.
- Keyboard avoiding view on all input screens.

[SECTION 7 — NOTIFICATIONS + STREAKS]
- Daily reminder via expo-notifications at user-set time.
- Streak-at-risk notification at 9pm if no entry today.
- Badge unlock notifications.
- Streak logic: increment if entry today AND yesterday, reset if gap, badge thresholds: [1,7,14,30,60,100].
- Quotes: fetch quotable.io, cache 10 in MMKV, fallback hardcoded 30, date-based daily rotation.
